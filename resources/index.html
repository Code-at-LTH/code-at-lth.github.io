Resources

A curated list of resources. Great if you want to learn something new!

Languages

  • Python - Dynamically typed popular language (Of Van Rossum, Dropbox, Google, Reddit et. al.)
  • Scala - Pragmatic language that runs on the JVM
  • Clojure - Modern Lisp dialect that runs on the JVM
  • C++ - Near-legendary language, long ago named "C with Classes", now it's far more
  • Go - Compiled and statically typed language by Google
  • Ruby - Of Ruby on Rails fame and much more!
  • Java - The language the ancient Java-course at LTH (and similar courses at many other schools) uses
  • Haskell - A purely functional language

  • Emacs
